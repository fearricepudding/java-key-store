<h3>Java keystore server</h3>

<p> 

keystore server. persistance on disk with a 1:2 file structure.

</p>

<h4>Methods</h4>

* GET
* PUT 
* DELETE


<h4>Running the server</h4>

 1. Download keystore release
 2. Unpack release
 3. make data writable
 4. Run server

``` bash
java -jar keystore.jar
```
